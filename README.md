# Parameters

## Dependencies

- cmake
- Boost

### Optional

- [termcolor](https://github.com/ikalnytskyi/termcolor)

### Install dependencies on macOS

```
brew install boost
```

## Build

This will only build the binary.

```bash
./bin/build.sh
```

Then the binary is available in `build_release`.

## Install

After building the project, run:

```bash
./bin/install.sh
```
